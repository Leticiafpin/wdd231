Site Plan

WDD 231 - Web Frontend Development

1. Site Name

Name: Local Travel Guide - Faro

Reason: A travel guide showcasing Faro’s history, culture, and attractions.

2. Site Purpose

This website serves as a comprehensive travel guide to Faro, Portugal, highlighting its hidden gems, historical sites, local traditions, and must-visit locations. It offers information on restaurants, outdoor activities, cultural events, and travel tips, along with an interactive map for easy navigation.

3. Scenarios

  • What are the best historical sites to visit in Faro?
  • Where can I find traditional Portuguese cuisine in Faro?

4. Color Scheme

Primary Color: Deep Blue (#003366) - used for headers and navigation elements.

Secondary Color: Warm Beige (#F5E1DA) - used for background and content sections.

5. Typography

Main Font: Montserrat (used in headings for a modern, elegant look)

Secondary Font: Open Sans (used in body text for easy readability)

6. Wireframe

A structured wireframe will display sections including an interactive map, blog, recommended places, and vibrant images.
